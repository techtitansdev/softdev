

# Getting Started

### Clone the repository
```bash
git clone 
```
### cd to app directory
```bash
cd gsi-iloilo
```
### Install Dependencies
```bash
npm i
```
# Development

### Set up env file
copy and paste the env from discord

### Set Up prisma
```bash
 npx prisma db push 
 ```

### To open Prisma Studio
```bash
 npx prisma studio 
 ```
### Run development
```bash
 npm run dev
 ```
